## 示例（先帮你填两条，剩下的你来加）

### 我想要：把两个 batch 的数据在 batch 维度拼起来
函数：`torch.cat([a, b], dim=0)`

例子：
```python
a = torch.randn(3, 4)
b = torch.randn(5, 4)
c = torch.cat([a, b], dim=0)   # shape: (8, 4)
```
踩过的坑：跟 `stack` 搞混——`cat` 不增加新维度，`stack` 会增加一个新维度。

### 我想要：把 loss 的 tensor 转成普通数字打印出来
函数：`.item()`

例子：
```python
print(f"loss = {loss.item():.4f}")
```
踩过的坑：直接 `print(loss)` 会打印一堆 tensor 元数据（dtype、grad_fn 等），不够干净。
