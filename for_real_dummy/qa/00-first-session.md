# Q&A — 第一次对话（2026-07-02）

## Q：我是大二本科，继承了博士学长的复杂仓库，怎么办？

**背景：** C 基础 + Python 基本语法，刚学完三大数学，无随机过程，无 PyTorch/DL 经验。

**Claude 的建议：**

1. **不要从头看整个仓库**——这是博士几年的积累，直接看会被淹没。
2. **选一条线进去**，从你最感兴趣的方向找入口。
3. **适合大二入门的起点**（按难度从低到高）：
   - Agent 基础（`learning/agent-foundations/`）—— 最易入门，核心是理解 LLM 的使用方式
   - LoRA 微调（`learning/lora-family/`）—— 需要一点 PyTorch
   - 强化学习基础（`learning/rl-foundations/`）—— 需要概率论，有一些数学
   - Transformer 原理（`learning/transformer-deep/`）—— 最硬核

4. **你已有的优势：**
   - C 基础 → 理解内存布局、指针、性能，看 GPU/CUDA 相关内容时有帮助
   - 三大数学 → 梯度（微积分）、矩阵乘法（线代）、期望/分布（概率论）在 ML 里到处都是
   - 随机过程没学 → **大多数 ML 论文用不到**，别担心

## Claude 的洞察

> C 语言的背景其实很有价值——你天然理解"内存在哪里"、"这个操作有多贵"，这是很多只学过 Python 的同学缺乏的直觉。PyTorch 的 GPU tensor 本质上就是"显存里的多维数组"，有 C 基础的人更容易建立这个心智模型。
