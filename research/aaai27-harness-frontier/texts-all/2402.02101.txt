A. Implementation Details
We implemented all LLM-based Prompt Optimization methods based on the APE repository 1 . During the initialization
stage, we utilized 4 initial examples for LLM-based prompt initialization and sampled a total number of 2 × 5 initial prompts
with 2 queries (each sample 5 prompts). For all Reflection-based methods (APO, PromptAgent, APO-Sum, and OPRO),
we sampled 4 error examples for the reflection process of each prompt. For other implementation details for OPRO, we
adhered to the default settings in the paper, which used a history record of 20 prompts arranged in ascending order for
prompt updating. We present all the used prompts and prompting cases for each method in Table 4 to Table 13.
We used GPT-42 for prompt initialization and prompt updating for all methods, with default hyperparameters except for
a temperature of 0.9. We utilized GPT-3.5-Turbo3 and Llama-2-70B-chat4 as target models, both setting the temperature
to 0 for inference, following previous work (Yang et al., 2023). To accelerate the inference of Llama-2-70B-chat, we
implemented it with the vLLM inference library (Kwon et al., 2023)5 .

Dateset Split The detailed statistics of the used datasets are shown in Table 3. We filtered 15 instances in snarks involving
the hate, sexual, violence, and self-harm categories that were detected by the OpenAI content filter 6 .
