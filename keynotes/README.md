# Keynotes Learning Library

This folder is a local archive of public keynote, invited-talk, workshop, and
researcher-entry pages collected for learning from top NLP/ML researchers.

- Main index and study method: [KEYNOTE_INDEX.md](KEYNOTE_INDEX.md)
- Source manifest: [sources.csv](sources.csv)
- Download audit log: [download_log.json](download_log.json)
- Local HTML snapshots: [source-pages/](source-pages/)

Scope of this first pass:

- Main conference keynote/invited-talk pages for ACL, EMNLP, CoLM, ICLR, ICML,
  and NeurIPS, focused on 2024-2025.
- Official workshop master lists for the same venues where available.
- A curated set of NLP/LLM/agents/safety/multilingual workshop pages that are
  likely to contain invited speakers, keynote talks, or strong problem-framing
  signals.
- Teacher-mentioned researcher entry: Xin (Eric) Wang.

One external page, `Visions of Language Modeling`, is retained as a URL but not
archived as HTML because Windows Defender flagged the downloaded snapshot. See
`source-pages/workshops/priority-pages/colm_2025_visions_lm.skipped.txt`.
