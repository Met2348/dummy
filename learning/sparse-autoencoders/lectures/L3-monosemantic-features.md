# L3 · 单义特征: 金门大桥与最大激活样本

> 22-min lecture · 目标: 理解怎么确认 SAE 特征单义 (最大激活样本), 及 Anthropic 的标志性成果。

---

## 0. SAE 提取了特征, 但它们单义吗?

L2 训出了 SAE, 得到许多特征。但怎么知道一个特征**真的单义** (对应一个干净概念)? 方法: **看它的最大激活样本** —— 哪些输入最点燃这个特征。如果都是同一概念 → 单义。

---

## 1. 最大激活样本: 给特征「贴标签」

```
   对特征 fi:
     跑大量输入 → 记录每个输入对 fi 的激活
     找激活最大的 top-k 输入 (token / 句子)
     看它们有没有共同主题:
       都是同一概念 → fi 单义 (如「都是金门大桥的提及」)
       五花八门     → fi 还多义 (没解干净)
```

> 这是给特征「贴人话标签」的标准方法。你 N2 会做: 对玩具 SAE 的每个特征, 找它最大激活的「当前值」—— 干净 SAE 的特征会**专一**对应某个值 (一个特征一个值)。真 gpt2 SAE 的特征会专一对应某类 token/概念。**最大激活样本 = 把特征从向量变成人能懂的概念。**

---

## 2. Anthropic 的「金门大桥特征」(标志性成果)

2024 年 Anthropic 在 Claude 上训了大规模 SAE, 提取出数百万个单义特征, 其中著名的:
- **金门大桥特征**: 一个特征专门对金门大桥的提及激活 (文字、不同语言、甚至图片描述)。
- **代码 bug 特征 / 谄媚特征 / 欺骗特征**等等。

更震撼的是**可干预**: 人为放大「金门大桥特征」, Claude 就满嘴金门大桥 (把自己当成金门大桥) —— 证明这个特征**因果地**控制相关行为 (接 M12.3 干预)。

> 这是 SAE 的里程碑: **在前沿大模型上提取出可理解、可干预的单义特征**。它支持了 mech interp 的核心信念 (M12.1: 网络内部有可读的 feature)。你的玩具 SAE 是这个的最小版: 提取单义特征 + 验证它对应一个概念。

---

## 3. 单义性的度量 (你 N1/N2 会算)

怎么定量「特征有多单义」? 几种:
- **纯度 (purity)**: 特征激活时, 主要集中在一个概念上的比例 (你 `sae.py` 的 `monosemanticity`)。高=单义。
- **最大激活样本的一致性**: top-k 激活样本是不是同一主题 (人工/自动判断)。
- **vs 原始神经元**: SAE 特征纯度 应 > 原始神经元纯度 (解叠加的收益)。

> 你 N1 会看到: **SAE 特征纯度 远高于原始神经元** (如 0.5+ vs 0.14) —— 这定量证明了 SAE 解了叠加。这是 SAE 价值的硬证据 (不是讲故事)。但注意 (L4): 纯度高 ≠ 特征「真实/有用」, 评估有争议。

---

## 4. 单义特征能干什么 (用途)

提取出单义特征后:
- **理解**: 模型内部有哪些概念 (特征字典 = 模型的「概念表」)。
- **监控**: 检测某概念是否激活 (如「欺骗特征」亮了 → 模型可能在骗, 接 M12.6)。
- **干预/控制**: 放大/抑制某特征, 精准调模型行为 (金门大桥 demo; 去偏见; 加安全)。
- **可解释的中间表示**: 用特征而非神经元描述 circuit (M12.5)。

> 这就是 SAE 为什么是「显微镜」: 它把不可读的叠加激活, 变成可读、可监控、可干预的概念字典。对**安全/对齐** (M12.6) 尤其有价值 —— 能监控「危险概念」是否激活。这是 SAE 火的原因。

---

## 5. 本讲小结 + 通往 L4

- **最大激活样本**: 找最点燃特征的输入, 看共同主题 → 给特征贴人话标签 (单义=同一概念)。
- **金门大桥特征** (Anthropic): 大模型上提取的可理解、**可干预**单义特征 (里程碑)。
- 单义度量: 纯度 / 最大激活一致性 / vs 原始神经元 (你 N1 看到 SAE 远超原始)。
- 用途: 理解 / 监控 (欺骗特征) / 干预控制 / 可解释 circuit (接 M12.5/12.6)。
