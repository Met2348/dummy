# L4 · 世界模型 × 具身 (通往 M11)

> 22-min lecture · 目标: 理解世界模型怎么给机器人当模拟器/规划器, 收口 M13.5, 把桥搭到 M11 具身。

---

## 0. 世界模型的「终点」: 给智能体用

L1-L3 建了世界模型 (学动态→想象→可交互)。它最终为谁服务? **决策智能体 (机器人/agent)**。世界模型是智能体的「内在模拟器」: 在脑内预演动作后果, 不靠昂贵的真实试错就能规划。

```
   智能体的两种学法:
   model-free: 直接在真环境试错 (贵, 危险, 慢)         ← 你 RL 模块的 PPO 等
   model-based: 先学世界模型, 在模型里"想象"着练/规划   ← 本专题, 省真实交互
```

---

## 1. 世界模型 + 规划 = model-based 决策

有了世界模型, 规划就是「在脑内搜动作序列」:
- **想象 rollout 评估动作**: 对每个候选动作序列, 用世界模型 imagine 出结果, 选最优 (你 N1 的 imagine 就是这步)。
- **MPC (模型预测控制)**: 每步都重新想象+重规划, 用真观测纠偏 (对抗 L1 的误差累积)。
- **在想象里做 RL** (Dreamer 系): 不在真环境、而在世界模型里跑 RL, 样本效率暴涨。

> 这把你 RL 模块 (model-free) 和本专题 (world model) 接起来: **世界模型 = RL 的「环境替身」**, 让智能体在想象里廉价地练。误差累积 (L1) 是这里的主敌, 所以要短视野 + 重规划 + 真观测纠偏。

---

## 2. 为什么具身智能特别需要世界模型 (接 M11)

机器人 (M11) 的真实数据**极贵极危险** (真机试错会摔坏、慢、不可逆)。世界模型是解药:
- **预演代替试错**: 机器人在世界模型里「想」一遍「这样抓会不会打翻」, 再真做。
- **sim2real 的世界模型版**: 学一个够准的世界模型当模拟器, 在里面训练策略 (接 M11 sim2real)。
- **数据增广**: 用世界模型生成想象轨迹, 补充稀缺的真机数据。

> 这就是 `world_model.py` 与 M11 **同源共享**的原因: M11 的 `world-action models` 专题会复用本专题的世界模型, 只是状态从 2D 点换成机器人观测 (图像+本体感觉)、动作换成关节指令。**机制不变, 接口升级。**

## 3. 谱系收束: 一切都在学「世界怎么动」

把课程几条线在这里收束:
| 线 | 学的是 | 与世界模型的关系 |
|---|---|---|
| 视频生成 (M13.4) | 生成逼真视频 | 隐式世界模型 (L2) |
| 扩散/transformer (M13) | 预测下一帧/token | 世界模型的「转移函数」实现 |
| RL (你的 RL 模块) | 决策策略 | 世界模型是它的环境替身 |
| 具身 (M11) | 机器人控制 | 世界模型当模拟器/规划器 |

> 大图景: **「学世界怎么从一个状态走到下一个」是生成、预测、决策、具身的共同内核**。世界模型是这个内核的显式名字。理解它, 你就看清了 M13 (生成) 和 M11 (具身) 为什么是一家。

## 4. 现状与开放问题 (gap, 接 M9 + capstone)

世界模型 2026 的 open 战场:
- **长程一致 + 持久状态**: 扩散续帧没记忆, 怎么记住远处场景 (L3)。
- **物理正确性**: 从统计近似到真物理 (L2 批判)。
- **可评测性**: 怎么证明世界模型「懂」而非「拟合」? (N2 量化误差累积是起点, 但远不够)。
- **真实尺度的动作条件**: 高维动作 (机器人多关节) 怎么准确条件化。
- **生成式 vs 解析式**: 端到端学 vs 显式物理引擎混合。

> 这些是你 capstone (M13.7) 和 M11 的 idea 富矿。用 M9.3 的「找 gap」框架扫这些方向, 每一个都能长出一个研究问题。

## 5. 本讲小结 (M13.5 收口) + 通往 M13.6

- 世界模型最终服务**决策智能体**: 当内在模拟器, 脑内预演省真实试错。
- **model-based 决策**: 世界模型 + 规划 (MPC/Dreamer), 接你 RL 模块 (世界模型=环境替身)。
- **具身特别需要它** (M11): 真机数据贵, 世界模型预演/当模拟器/数据增广; 故 `world_model.py` 与 M11 同源。
- 大图景: 「学世界怎么动」是生成/预测/决策/具身的共同内核; 世界模型是它的显名。
- gap: 长程记忆、物理正确、可评测、高维动作 (capstone + M11 富矿)。
