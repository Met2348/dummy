# L1 · 什么是世界模型: 学环境的动态

> 24-min lecture · 目标: 理解世界模型 = 学「状态+动作→下一状态」的转移函数, 学会后能在脑内想象未来。接 M13.4 视频, 通 M11 具身。

---

## 0. 从「生成视频」到「预测世界」

M13.4 的视频生成是「无条件/文本条件地生成一段会动的画面」。世界模型多问一句:
**「如果我现在做动作 a, 下一刻世界会变成什么样?」**

```
   视频生成:        噪声 / 文本  → 一段视频
   世界模型:  当前状态 + 动作 a   → 下一个状态 (再接着预测下下个…)
```

> **世界模型 = 学环境的转移动态** $P(s_{t+1} \mid s_t, a_t)$: 给当前状态和动作, 预测下一个状态。
> 它把「生成」变成「**带动作条件的预测**」。这一步之差, 让模型从「会做梦」变成「能推演后果」。

---

## 1. 核心能力: 在脑内「想象」(imagination)

世界模型一旦学会转移函数, 就能**不碰真环境**、只在脑内链式预测一整条未来轨迹:

```
   s0 --(a0)--> ŝ1 --(a1)--> ŝ2 --(a2)--> ŝ3 ...   (全是模型预测, 没碰真环境)
```

这叫 **imagined rollout (想象 rollout)**。它的价值:
- **规划/做梦**: 在脑内试很多动作序列, 挑结果最好的去真执行 (model-based 规划)。
- **样本效率**: model-based RL 在世界模型里「练」, 省下昂贵的真环境交互 (接你 RL 模块)。
- **机器人预演**: 机器人在世界模型里预演「抓这个杯子会怎样」再动手 (接 M11)。

> 你的 N1 会亲手做: 训一个世界模型, 给它起点+一串动作, 让它**想象**整条轨迹, 再和真环境对照。你会看到想象的轨迹紧贴真实 —— 模型真的学到了「世界怎么动」。

---

## 2. 为什么预测「变化量」而非「绝对状态」

实现细节 (但是重要直觉): `world_model.py` 让模型预测 **Δstate (变化量)** 而非直接预测下一状态:

```
   ŝ_{t+1} = s_t + model(s_t, a_t)      # 预测增量 Δ, 加回当前状态
```

> 为什么? 因为「下一状态」大部分等于「当前状态」(世界有惯性), 让模型只学**差异**更稳、更易学。这和扩散预测噪声 (M13.1)、ResNet 学残差是同一思想: **让模型学「变化/残差」而非「全量」**。同一个工程智慧反复出现。

---

## 3. 世界模型的核心难题: 误差累积 (compounding error)

想象 rollout 是**自回归**的 (用上一步预测当下一步输入)。问题: 每步的小误差会**喂进下一步**, 复利式放大:

```
   1 步:  误差小 (模型准)
   10 步: 误差中 (前面的误差累积)
   20 步: 误差大 (复利放大, 想象轨迹逐渐偏离真实)
```

> 你的 N2 会量化这条**误差增长曲线**: 1 步预测很准, 但 20 步后想象明显偏离。这是世界模型 (和一切自回归预测、L3 长程视频一致性) 的**根本难题**: 短期可信、长期发散。缓解靠: 更准的模型、更短的规划视野、周期性用真观测「纠偏」(接 RL 的 MPC/重规划)。

---

## 4. 世界模型的谱系 (从你学过的连过来)

世界模型不是新东西, 你学的好几块都是它的零件:
- **转移函数** = 你 RL 模块的环境模型 (model-based RL 的「model」)。
- **观测可以是高维的**: 状态不一定是 2D 点, 可以是**整帧图像** → 这时世界模型就是「带动作条件的视频预测」(L2)。
- **生成式世界模型**: 用扩散/transformer 当转移函数, 预测下一帧/下一 latent (接 M13.4)。

> 关键认知: **世界模型 = 动作条件的生成式预测**。把 M13.4 的视频生成加上「动作条件」, 就从「做梦」升级成「可推演的世界模拟器」。这正是 L2 要讲的 2024 洞察。

---

## 5. 本讲小结 + 通往 L2

- 世界模型 = 学转移动态 $P(s_{t+1}\mid s_t,a_t)$: 给状态+动作, 预测下一状态。
- 核心能力 **imagined rollout**: 脑内链式预测未来, 用于规划/做梦/省样本/机器人预演。
- 实现智慧: 预测**变化量 Δ** 更稳 (同扩散学噪声/ResNet 学残差)。
- 核心难题 **误差累积**: 自回归预测短期准、长期发散 (通 L3 长程视频、RL 重规划)。

> **下一讲 L2「视频模型当世界模拟器」**: 2024 的大洞察 —— 把状态换成**整帧图像**, 视频生成模型 (Sora) 本身就是一个隐式世界模型 (它学会了物体怎么动/碰撞)。L2 讲这个范式转变。
