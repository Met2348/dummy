# L1 · 文本也能扩散: masked diffusion LM

> 24-min lecture · 目标: 理解 masked diffusion 怎么把扩散搬到离散文本, 以及并行迭代解码。接你 AR 本行, 通 M13.1 连续扩散。

---

## 0. 一个挑衅式的问题

你的本行是自回归 LLM (AR): 左到右, 一次预测一个 token。扩散 (M13.1-13.5) 是在**连续空间**加噪去噪。文本是**离散** token, 怎么扩散? 答案是 **masked diffusion (离散/吸收态扩散)**:

> **dLLM 的前向过程 = 逐步把 token 替换成 [MASK]** (而非加高斯噪声); **反向 = 学着把 [MASK] 填回**。生成时从「全是 [MASK]」出发, **并行迭代地**把它们填成文本。代表: LLaDA、早期的 D3PM/MaskGIT(图)。

---

## 1. 前向: 加噪 = 逐步遮盖

连续扩散 (M13.1) 前向是「逐步加高斯噪声直到纯噪声」。离散文本的对应物:

```
   连续 (M13.1):  清晰图 → 加噪 → ... → 纯高斯噪声
   离散 (dLLM):   完整文本 → 遮盖一些 → ... → 全是 [MASK]
```

- 「噪声水平」= **遮盖比例** r: r=0 是原文, r=1 是全 [MASK]。
- 训练时对每个样本随机抽一个 r, 按 r 遮盖, 让模型预测被遮位的原 token。

> 这就是 BERT 的 masked LM, 但**遮盖比例从固定 15% 变成 0~100% 全谱**, 并解释成一个扩散过程。你的 N1 训练循环就是: 随机 r → 遮盖 → 只在被遮位算交叉熵。**dLLM ≈ "全比例" 的 BERT + 迭代解码。**

---

## 2. 反向: 去噪 = 学填被遮位

模型是一个**双向 transformer** (无因果掩码!), 输入含 [MASK] 的序列, 每个位置输出 V 个 token 的 logits:

```
   输入:  [a, MASK, c, MASK, b, a]
   模型:  双向看全文 (含两侧已知)
   输出:  每个 MASK 位的 token 分布
```

> 关键: **双向**。AR 用因果掩码 (只能看左边); dLLM 没有因果掩码, **每个位看左右全文**。这一点差异是 dLLM 一切优势 (双向 infilling, L3) 的根源。你的 `build_dlm` 就是个无掩码的 TransformerEncoder。

---

## 3. 生成: 并行迭代解码 (非自回归)

生成不是「一次填光所有 [MASK]」(一次填光质量差, 因为各位独立预测会不一致)。而是**迭代**, 每轮提交最有把握的:

```
   轮0: [M, M, M, M, M, M]            全 MASK
   轮1: [M, M, M, 3, b, M]            填最自信的 2 位
   轮2: [a, M, M, 3, b, a]            再填 2 位 (现在能参考已填的)
   轮3: [a, b, 3, 3, b, a]            填完
```

机制 (`generate_dlm`): 每轮 → 模型预测所有 [MASK] 位 → 按**置信度**排序 → 提交最自信的 k 个 → 其余留 [MASK] 下轮再填。这叫 **confidence-based parallel decoding** (MaskGIT 路线)。

> 妙处: **一轮能填多位 (并行)**, 所以 T 轮 (T < L) 就能生成长度 L 的序列, 而 AR 必须 L 步。你的 N1 会看到这条**并行解码轨迹**: 从全 [MASK] 几轮填成完整文本, 不是左到右。

---

## 4. 并行度 vs 质量: dLLM 的核心旋钮

填得越「猛」(每轮填越多 → 轮数越少) 越并行, 但质量越差 (同轮内多位独立预测易冲突)。填得越「稳」(每轮填 1 位 → 轮数 = L) 质量越好, 但退化成顺序解码:

```
   轮数 = 1:  全并行 (最快), 质量最低
   轮数 = L:  顺序 (最慢), 质量最高
```

> 这**完全对应扩散的「采样步数 vs 质量」(M13.2)**: 少步快但糙, 多步慢但精。dLLM 把这个旋钮带到了文本生成 —— 你 N1 会画出这条「轮数 vs 合法率」曲线, 亲眼看权衡。这也是 dLLM 加速研究 (少步高质量解码) 的战场, 和 M13.2 同源。

---

## 5. 本讲小结 + 通往 L2

- dLLM 前向 = **逐步遮盖 token** (噪声=遮盖比例); 反向 = **双向 transformer 学填被遮位** (≈全比例 BERT)。
- 生成 = 从全 [MASK] **按置信度并行迭代解码** (非自回归, T<L 轮)。
- **双向** (无因果掩码) 是 dLLM 一切优势的根源。
- 并行度 vs 质量旋钮 = 扩散步数vs质量 (M13.2) 的文本版。
