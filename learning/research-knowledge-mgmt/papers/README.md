# papers/ — research-knowledge-mgmt 参考源

这个专题是**方法/工作流**类, 不绑定某篇 NLP 论文, 所以 papers/ 收的是「研究方法 + 知识管理」的经典参考。

## 知识管理方法论
- **How to Take Smart Notes** — Sönke Ahrens (2017). Zettelkasten 方法的现代普及本, L2 的思想来源。中文版《卡片笔记写作法》。
- **Building a Second Brain** — Tiago Forte (2022). "第二大脑" + CODE/PARA 方法; 本专题标题即来自此概念。
- Niklas Luhmann 的 Zettelkasten 原始实践 (社会学家, 一生 70 本书 / 400 篇论文)。搜 "Luhmann Zettelkasten" 有大量二手解读。

## 为什么 papers/ 这么轻
本专题的「知识」在**可跑的 src 工具** (`bib_to_cards` / `arxiv_triage`) 和**课件里的工作流**中, 不在论文 PDF。上面的书/文档用于深读与照着配工具。

> 用 9.3 `critical-reading-gap` 的批判式阅读读这些方法论书: 它们大多是经验之谈而非实证研究, 别全信, 挑适合你的拿、不适合的丢。
