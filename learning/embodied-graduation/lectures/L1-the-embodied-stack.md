# L1 · Module 11 全栈回顾: 一张图串起具身

> 22-min lecture · 目标: 把 M11 六个专题串成一条线, 提炼贯穿全栈的核心信念。Capstone 开篇。

---

## 0. 你这一路造了什么

M11 从「不懂具身」走到「能拆解并组装 VLA + 用世界模型规划 + 弥合 sim2real」。回头看全栈:

```
  11.1 地基      tokens-as-actions (动作当token, 控制=next-token), toy_env 共享地基
   │
  11.2 VLA架构    backbone (VLM) + 动作头; 离散 vs 连续
   │
  11.3 扩散动作头  多峰 → 扩散/flow (接 M13); chunking 权衡
   │
  11.4 数据/模仿   BC + 分布漂移 + 数据 scaling + 人类视频
   │
  11.5 世界模型    学转移 → 想象 → MPC 规划 (接 M13.5); model-based
   │
  11.6 sim2real   仿真训练 + sim2real gap + DR (你的 IsaacLab/AntBot)
```

> 不是六个孤立专题, 是**一条从「怎么表示动作」到「怎么训」到「怎么部署真机」的完整链路**。每一步都复用你 M10 (VLM) / M13 (扩散/世界模型) / RL 的地基。

---

## 1. 贯穿全栈的核心信念 (三条)

**信念一: 具身 = 你已学能力 + 动作 + 闭环。**
- VLA = VLM 感知核 (M10) + 动作头 + 机器人数据。你已懂感知和 scaling, 具身补「动作」。

**信念二: 控制 = 预测 / 生成 / 规划 (你都学过)。**
- tokens-as-actions: 控制 = next-token (你的 LLM)。
- 扩散动作头: 控制 = 条件生成 (你的 M13)。
- 世界模型规划: 控制 = 想象 + 搜索 (你的 RL + M13.5)。

**信念三: 机器人的核心瓶颈是数据 + sim2real。**
- 数据贵 (M11.4) → scaling/视频/仿真; sim≠real (M11.6) → DR。这是具身区别于 NLP 的两大特殊性。

> 带着这三条信念, 你能拆解任何 VLA/具身系统: 问「感知核是什么 / 动作怎么生成 / 数据从哪来 / 怎么弥合 gap」。

## 2. 全栈的「复用」: M11 站在你已学之上

| M11 专题 | 复用了你的 |
|---|---|
| 11.1 tokens-as-actions | LLM next-token (M1-M3) |
| 11.2 VLA backbone | VLM (M10) |
| 11.3 扩散动作头 | 扩散/flow (M13.1/13.2) + 条件 (M13.3) |
| 11.4 数据 co-train | 数据治理 + 冻结 (M10.3) |
| 11.5 世界模型 | M13.5 世界模型 + 你的 RL |
| 11.6 sim2real DR | 分布偏移/泛化 (ML 基础) |

> 看这张表: **M11 几乎没有"全新"的东西, 全是你已学能力嫁接到物理世界**。这就是为什么你 (NLP/LLM 背景) 能快速转具身: 地基 80% 是共享的。capstone 的价值就是让你看清这个「复用」。

## 3. 你现在站在哪 (能力盘点)

走完 M11, 你应该能:
- 解释 tokens-as-actions, 把控制变 next-token (11.1)。
- 拆 VLA = backbone + 动作头, 选离散/连续/扩散 (11.2/11.3)。
- 用 BC 训策略, 理解分布漂移 + 数据 scaling (11.4)。
- 用世界模型想象 + MPC 规划 (零专家), 对比 model-based/free (11.5)。
- 用 DR 弥合 sim2real gap, 跑通 IsaacLab (11.6)。

> 这是一份能在面试/科研里直接用的「具身/VLA」能力清单。N1 会让你**亲手把全栈跑一遍** (mini-benchmark: 多方法在 toy 上比成功率)。
