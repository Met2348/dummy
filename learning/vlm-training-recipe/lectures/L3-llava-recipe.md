# L3 · LLaVA 配方拆解: 照着训一个真 VLM

> 25-min lecture · 目标: 把 L1/L2 的原则落到 LLaVA 这个经典、可复现的配方上 —— 一步步看一个真 VLM 怎么训出来。

---

## 0. 为什么拆 LLaVA

LLaVA 是开源 VLM 的「标准配方」: 简单、可复现、效果好、被无数工作当 baseline。拆透它, 你就有了训任何 VLM 的模板。它正好把 L1 (两阶段) + L2 (数据/冻结) 全用上。

---

## 1. LLaVA 的组件 (全是现成的)

```
   视觉塔: CLIP ViT-L/14 (冻结)        ← M10.1 的对比预训练视觉塔
   连接器: 一个 MLP (LLaVA-1.5)        ← M10.2 的投影路线
   LLM:    Vicuna / LLaMA (预训练)     ← 你已有的纯文本 LLM
```

> 关键: **三块里两块 (视觉塔 + LLM) 都是别人训好的现成品, 直接拿来。** LLaVA 真正「造」的只有那个 MLP 连接器 + 对 LLM 的微调。这就是它便宜可复现的根本 —— 站在两个强预训练模型的肩上。

---

## 2. 阶段 1: 特征对齐预训练

```
   数据: ~558K 图文对 (从 CC/LAION 等筛的「图 + 一句描述」)
   任务: 给图, 生成它的描述 (captioning)
   训:   仅 MLP 连接器 (视觉塔冻、LLM 冻)
   时长: 相对短 (只训小 MLP)
```

这一步让 MLP 学会把 CLIP 视觉 token 投影成「LLM 读得懂的视觉 token」。结束后, 模型会看图说话, 但不会按指令做事。

> 对应你 N1 的合成任务: 连接器学会把「图的类别信息」传给 LLM。虽然简化, 但机制一样 —— **连接器是视觉和语言之间唯一需要从头学的桥**。

---

## 3. 阶段 2: 视觉指令微调

```
   数据: ~150K 视觉指令数据 (LLaVA 用 GPT-4 看图描述/框, 合成「图+多样指令+答案」)
   任务: 按指令处理图 (描述/推理/对话/定位)
   训:   MLP 连接器 + LLM (解冻 LLM); 视觉塔仍冻
   时长: 中等
```

两个细节:
- **指令数据是合成的**: LLaVA 的创新之一是用 GPT-4 (当时纯文本, 喂它图的标注) **批量生成**高质量视觉指令数据。这解决了「真实视觉指令数据稀缺」的问题 —— 是合成数据 (你 data-curation 专题) 在 VLM 的经典应用。
- **解冻 LLM**: 让 LLM 真正学会「用视觉信息执行指令」, 不只是被动接收。

结束后, 就是一个能对话、能回答图像问题的 VLM。

---

## 4. LLaVA 配方的可复现性 (对你的意义)

LLaVA 配方对博0 极友好:
- **组件现成**: CLIP + LLaMA + 一个 MLP, 都开源。
- **数据公开**: 对齐数据和指令数据都放出来了。
- **算力可控**: 因为只训 MLP + 微调 LLM (可用 LoRA, 你 M1 PEFT 的知识!), 单机几张卡能跑。
- **是强 baseline**: 你的 VLM 研究可以从复现 LLaVA 起步, 然后改一个组件做消融 (接 9.3 找 gap: LLaVA 的某个设计是不是最优? 哪个连接器/哪层视觉特征更好?)。

> 给造 VLM 的人: **第一个 VLM 就复现 LLaVA。** 它是 VLM 界的「R1-Zero」——一个干净、可复现、能当研究起点的基线。复现它, 你就走通了「造 VLM」的完整流程, 后面改什么、消融什么都有了地基。

---

## 5. 从 LLaVA 到前沿 (一句话)

LLaVA 之后的演进 (LLaVA-1.5/NeXT、Qwen-VL、InternVL 等) 大多在改:
- **更高分辨率** (动态分辨率 / 切图, 缓解 token 爆炸, 接 M10.2 的上下文成本)。
- **更好的视觉塔** (SigLIP / 双塔, M10.1)。
- **更多更好的指令数据** (规模 + 质量)。
- **更强的 LLM 底座**。

但核心两阶段配方不变。理解了 LLaVA, 你就理解了这一整条线。

---

## 6. 本讲小结 + 通往 L4

- LLaVA = 冻结 **CLIP 视觉塔** + **MLP 连接器** + 微调 **LLaMA/Vicuna**; 两块现成, 只造一座桥 + 微调。
- 阶段 1: 图文对, 只训 MLP (对齐); 阶段 2: **合成的**视觉指令数据 (GPT-4 生成), 训 MLP + LLM。
- 可复现、算力可控 (可配 LoRA)、是强 baseline —— **博0 造 VLM 的最佳起点**。
- 后续演进改分辨率/视觉塔/数据/底座, 但两阶段配方不变。

> **下一讲 L4「训练陷阱」**: 配方对了也可能训崩 —— 模态坍缩 (只看文不看图)、灾难遗忘 (微调后不会纯聊天)、数据配比失衡。L4 讲这些坑和怎么避。

**动手**: 在 N1 训完 mini-VLM 后, 对照 LLaVA 配方写出「如果我要复现真 LLaVA, 三块组件各用什么、两阶段各用什么数据、冻结怎么设」—— 把配方变成你自己的 checklist。
