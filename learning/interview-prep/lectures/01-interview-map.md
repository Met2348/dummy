# L01 · 研究岗面试全景 + 你的 70/30 配比

## 你不缺第 86 个专题

85 专题的**广度**早过了门槛。面试考的是另一件事——把知识在**限时 / 口头 / 真框架**下演示出来。这是**模式切换**,不是内容补课。

## 研究岗(RS/RE)面试 loop 的真实构成

| 轮次 | 考什么 | 你的现状 | 本 track 对应 |
|------|--------|---------|--------------|
| **ML coding** | 手写 attention / 训练循环 / 采样,真 torch | 全 mock | `mlcoding/` ✅ |
| **算法 coding** | LeetCode medium(研究岗偏轻但有) | 零 | `leetcode/` ✅ |
| **ML/DL 基础** | rapid-fire 秒答 | EE 可能有洞 | `mlqa/` ✅ |
| **系统设计** | ML system design(研究向:训练/评测 pipeline) | 有素材没格式 | L06 |
| **研究面试** | 讲论文 / 批判 / 提实验 / 走自己工作 | Module 9 打底 | L07 + hero project |
| **behavioral** | 项目 impact / STAR | 自学难讲 | L08 |

> 关键认知:**研究岗的 coding 是"保险"不是"主战场"**——但它是**二元过滤**,挂一轮直接出局。所以策略是"够用地板",不是刷成竞赛选手。

## 你的 70/30 配比

- **研究 70%**:hero project(真复现出真数字)+ 论文口头 drill + 开放式科研推理。矛。
- **coding 保险 30%**:真 torch 从零(与研究肌肉双用)+ 15 pattern 地板 + 基础秒答。盾。

## 时间线(2026.7 起)

博士 2026 秋开学 → 暑研招聘多在开学后当年秋冬 → 第一波真面试约 2026 末–2027 初。**跑道够,现在启动正合适**。

分配建议(每周):
- 研究:1 个 hero-project 推进 + 2 篇论文能上白板讲。
- coding:3–4 道 LeetCode(按 15 pattern 轮)+ 1 个 mlcoding 模块闭卷默写。
- 基础:每天 5 题 rapid-fire(用 `mlqa.grade` 自评)。

## 最容易被忽视的杀手:口头表达

强自学者常"脑子懂、嘴上乱"。你的产出全是文字/单人,面试是口头/社交。**录音练讲、找人 mock** 比再读一篇论文重要。每个 mlcoding 模块都练一遍"边写边讲 shape 和为什么"。
