# L1 · 为什么用仿真 + sim2real gap

> 22-min lecture · 目标: 理解仿真训练的价值 (安全/便宜/可并行) 和它的核心问题 (sim2real gap)。Module 11.6 开篇。

---

## 0. 真机训练太贵, 仿真是出路

M11.4 说真机数据贵。更彻底的省钱方式: **在仿真里训练**。仿真器 (如 NVIDIA Isaac Sim / IsaacLab) 能模拟机器人+物理, 让策略在虚拟世界里练, 不碰真机。

```
   真机训练:  慢, 贵, 危险 (会摔坏), 难并行
   仿真训练:  快 (并行几千个环境), 便宜, 安全 (摔了重置), 可大规模 RL
```

> 仿真把「机器人 RL」从「不可行 (真机太贵)」变成「可行」。你 M11.5 的 model-based RL、标准 model-free RL, 大多在仿真里跑。**仿真是机器人能用 RL 的前提。**

---

## 1. 仿真的三大优势

- **安全**: 摔了/撞了就重置, 不损失硬件 (真机摔一次可能几万块)。
- **便宜 + 快**: 没有真实时间约束 (能比实时快), 不用人盯着。
- **可并行**: 一块 GPU 能并行几千个仿真环境 (IsaacLab 的卖点), RL 样本暴增 → 训练快几个数量级。

> 这就是为什么 Isaac Sim/IsaacLab 这类 **GPU 并行仿真器** 火: 它们把成千上万个机器人环境塞进 GPU 并行跑, 让 RL 的「样本饥渴」(你 RL 模块知道的) 在仿真里被满足。**并行仿真 = 机器人 RL 的发动机。**

---

## 2. 但: sim2real gap (核心问题)

仿真不是真实。**在仿真里完美的策略, 搬到真机会掉点** —— 这就是 **sim2real gap**:

```
   sim 里完美 ──部署──> real 里掉点
   因为: sim ≠ real (物理近似/传感器/未建模因素都有差异)
```

gap 的来源:
- **物理差异**: 仿真的摩擦/接触/动力学是近似的, 和真实不完全一样。
- **传感器差异**: 仿真的相机/传感器太"干净", 真实有噪声/畸变/延迟。
- **未建模因素**: 真实世界有仿真没建的东西 (风/磨损/光照变化)。
- **配置覆盖不足**: 仿真只覆盖了部分场景, 真实更多样 (你 N1 会量这个)。

> sim2real gap 是仿真训练的**头号敌人**: 不解决, 仿真里再好也白搭。它本质是**训练分布 (sim) ≠ 测试分布 (real)** —— 和你 M11.4 BC 的分布漂移、机器学习的分布偏移是同一类问题。

---

## 3. gap 的本质: 分布偏移 (接 M11.4)

把 sim2real gap 看成分布偏移, 就和你已学的连上了:
```
   BC 分布漂移 (M11.4): 训练状态分布 ≠ 测试状态分布
   sim2real gap:        sim 分布 ≠ real 分布
   → 同一个根: 模型只在见过的分布上可靠
```

> 你 N1 会用一个 GPU-free 玩具坐实这点: 在**窄**配置 (sim) 训的策略, 到**全**配置 (real) 掉点 —— 因为它没见过那些配置。gap = 覆盖不足。**解法的方向就清楚了: 让 sim 的训练分布覆盖 real** —— 这就是 domain randomization (L3)。

---

## 4. 弥合 gap 的两条路 (L3/L4 展开)

- **domain randomization (L3)**: 训练时随机化仿真参数 (摩擦/光照/噪声/配置), 让策略见过足够多变化 → 对真实的未见变化鲁棒。**主力方法。**
- **更真的仿真 (sim fidelity)**: 把仿真做得更像真实 (更准物理/渲染)。有用但有上限 (永远不完全等于真实)。
- **真机微调**: 仿真预训练 + 少量真机数据微调 (接 M11.4 预训练+微调)。

> 主线是 **domain randomization**: 与其追求「完美仿真」(做不到), 不如「随机化到覆盖真实」。这是 sim2real 最实用的智慧 (L3 详讲, N1 实测)。

---

## 5. 本讲小结 + 通往 L2

- **仿真训练**: 安全/便宜/可并行 (IsaacLab 并行几千环境), 是机器人 RL 的发动机。
- **sim2real gap**: sim 里好、real 里掉点; 来源 = 物理/传感器/未建模/覆盖不足。
- gap 本质 = **分布偏移** (sim≠real), 同 M11.4 分布漂移; 模型只在见过的分布可靠。
- 弥合两路: **domain randomization (主力)** / 更真仿真 / 真机微调。
