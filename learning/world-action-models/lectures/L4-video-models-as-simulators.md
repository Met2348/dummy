# L4 · 视频模型当模拟器 (接 M13)

> 20-min lecture · 目标: 把世界模型升到图像观测 —— 视频生成模型当机器人模拟器。收口 11.5, 焊接 M13.4/13.5。

---

## 0. 从「2D 状态」到「图像观测」

L1-L3 的世界模型用低维状态 (toy 的 agent+goal)。真实机器人的观测是**图像**。这一讲把世界模型升到图像: **视频生成模型 (M13.4) 就是图像空间的世界模型** —— 给当前帧+动作, 预测下一帧。

```
   低维世界模型 (L1-L3):  state, action → next state
   视频世界模型 (本讲):   当前帧, 动作 → 下一帧  ← 就是动作条件的视频生成 (M13.4/13.5-L3)!
```

---

## 1. 视频模型 = 图像空间的世界模型 (接 M13.5-L2)

回到 M13.5-L2 的洞察: **视频生成模型为了生成逼真视频, 被迫学了世界的物理**, 所以是隐式世界模型。加上**动作条件** (M13.5-L3 可玩世界), 它就成了机器人能用的图像世界模型:

```
   动作条件视频模型: 过去帧 + 机器人动作 → 预测未来帧
   = 机器人在图像空间的"想象": "我这样动, 看到的画面会怎样变"
```

> 这就是 M13.4 (视频生成) + M13.5 (世界模型) 在机器人上的落地: **视频世界模型让机器人在图像空间想象动作后果**, 再规划 (L1 MPC) 或在想象里 RL (L3)。GR00T 等就用这个思路。

---

## 2. 为什么图像世界模型这么有价值

- **直接用真实观测**: 机器人看的就是图像, 在图像空间想象最贴近真实。
- **复用视频生成的全部进展**: M13.4 的时空建模、latent、少步采样全用得上。
- **海量视频可预训练** (M11.4-L4): 人类视频/网络视频学图像动态, 再接动作。

> 这把你 M13 的全部生成技术接到了机器人: 视频扩散 (M13.4)、世界模型 (M13.5)、少步采样 (M13.2, 实时性) 全是图像世界模型的零件。**M13 不是孤立的生成课, 它是具身世界模型的技术库。**

---

## 3. 硬难题 (gap, 接 M13.4-L3)

图像世界模型继承了视频生成的所有难题, 在机器人上更尖锐:
- **长程一致** (M13.4-L3): 想象长了场景漂移 → 限制规划视野。
- **物理正确性** (M13.5-L4): 想象的物理不对 → 规划基于错误想象。
- **实时性** (M13.4-L4): 机器人要高频, 视频模型推理慢 → 需少步采样 (M13.2)。
- **动作-后果对齐**: 动作条件要准 (M13.5-L3)。

> 这些是 model-based 具身的最前沿 open 问题, 每个都连着你学过的 gap。对你 (找方向): **「准确 + 实时 + 长程一致的图像世界模型」是具身的圣杯之一**, 缺口大、价值高, 且复用你 M13 全部地基。
