# L1 · VLA 的两阶段结构: 感知核 + 动作头

> 24-min lecture · 目标: 理解 VLA = VLM backbone (感知/推理) + action head (动作解码) 的两阶段结构。Module 11.2 开篇, 接 M10。

---

## 0. VLA 长什么样

M11.1 说 VLA = VLM + 动作头 + 数据。这一讲拆开「VLM + 动作头」, 看清 VLA 的标准**两阶段结构**:

```
   观测 (图像 + 指令 + 本体感觉)
        │ ① 感知/推理 backbone  ← 真实里是 VLM (你的 M10)
   特征 (理解了"看到什么、要干什么")
        │ ② 动作头 (action head/decoder)
   动作 (关节/末端/离散 token)
```

> **VLA = 两阶段: backbone (理解世界) + 动作头 (决定动作)。** 这个分工是几乎所有 VLA 的共同骨架。理解它, 你就能拆解任何 VLA (OpenVLA/π/GR00T 都是这个结构, L2)。

---

## 1. 阶段①: 感知/推理 backbone (= VLM)

backbone 负责**理解**: 看图像、读指令、(可选) 读本体状态, 输出一个富含语义的特征。

- 真实 VLA: backbone **就是一个 VLM** (你的 M10) —— 视觉编码器 (M10.1) + 投影 (M10.2) + LLM。
- 它带来 RT-2 的涌现泛化: VLM 的世界知识 (物体/空间/常识) 让机器人泛化 (M11.1-L2)。
- 你的 `mini_vla.py` 里 backbone 是个小 MLP (玩具), 但**位置和角色和真实 VLM 完全一样** —— 它是「VLM 插槽」。

> 关键认知: **backbone 阶段是你 M10 的全部内容**。VLA 没有重新发明感知, 它直接用 VLM。所以你懂 M10 = 懂 VLA 的一半 (感知那一半)。

---

## 2. 阶段②: 动作头 (action head)

动作头负责**决定**: 把 backbone 的特征解码成可执行的动作。

动作头有几种 (L3 深入, M11.3 再深入):
- **离散 token 头**: 输出动作 token 的分类分布 (RT-1/RT-2)。简单, 复用 LLM 的输出层。
- **连续回归头**: 直接回归连续动作值。精度高, 但难表达多峰。
- **扩散/flow 头**: 用扩散生成动作 (π, M11.3)。多峰 + 平滑 + 精细。

> 你的 `mini_vla.py` 提供**可换动作头** (discrete / continuous), N2 会对比它们。这就是 VLA 设计的核心自由度: **backbone 大同小异 (都是 VLM), 区别和创新主要在动作头**。M11.3 整个专题就在讲最强的动作头 (扩散)。

---

## 3. 为什么这样分工 (解耦的好处)

两阶段解耦带来工程上的巨大好处:
- **复用预训练**: backbone 用预训练 VLM (不用从零学感知), 只需训/微调动作头 + 轻调 backbone。
- **模块化**: 换更强的 VLM → 整个 VLA 变强; 换更好的动作头 → 控制更精细。互不干扰。
- **冻结策略**: 可以冻 backbone (省算力/防遗忘) 只训动作头, 或联合微调 (接 M10.3 的冻结/解冻)。

> 这和你 M10.3 的「VLM 训练食谱」(冻视觉/冻 LLM/联合微调) 完全一脉相承。VLA 训练 = M10 训练 + 一个动作头。你 N1 会组装这个结构, 体会「backbone + head」的解耦。

---

## 4. 把 M10 接进来 (你已造好一半)

回看你 M10.3 造的 mini-VLM (`mini_vlm.py`): 它已经是一个完整的 backbone! VLA 只需在它后面接动作头:

```
   M10.3 mini-VLM (视觉编码 + 投影 + LLM)  =  VLA 的 backbone 插槽
                                          + 动作头 (本专题)
                                          =  mini-VLA
```

> 你 N1 会**真的 import M10 的 mini_vlm**, 确认它就是 VLA 的感知核; 然后在玩具控制任务上用「backbone + 动作头」结构组装 mini-VLA。**你不是从零造 VLA, 你是给已造好的 VLM 接一个动作头。**

---

## 5. 本讲小结 + 通往 L2

- VLA = 两阶段: **backbone (VLM, 理解) + 动作头 (解码动作)**, 几乎所有 VLA 的共同骨架。
- backbone 阶段 = 你 M10 的全部 (VLA 不重发明感知, 直接用 VLM, 带来涌现泛化)。
- 动作头是核心自由度: 离散 token / 连续回归 / 扩散 (L3 + M11.3); 创新主要在这。
- 解耦好处: 复用预训练 + 模块化 + 冻结策略 (接 M10.3)。
