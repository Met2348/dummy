## 0. 评测也能骗人 (VLM 版)

你在 9.3 学过攻击别人的论文, 9.6 学过图的诚实。VLM 评测有一批**特有陷阱**, 既是你审别人 VLM 论文时要查的 (9.3), 也是你自己报 VLM 结果时要避的 (9.5/9.6)。

---

## 1. VLM 评测的特有陷阱

| 陷阱 | 怎么骗 | 怎么避 (接的专题) |
|---|---|---|
| **benchmark 污染** | 报的 benchmark 图被训练见过 | 用新/私有 benchmark; 报训练数据来源 (9.3 Q4) |
| **挑有利样例** | 论文里展示的定性样例是 cherry-picked | 标明是否 cherry-picked, 给随机样例 (9.6-L4) |
| **不报幻觉** | 只报准确率, 不报 POPE/幻觉 | 必报抗幻觉指标 (L2/L3) |
| **单一总分掩盖偏科** | 一个均分藏住「OCR 极弱」 | 报多维剖面 (L1/L2, 9.6 一图一信息) |
| **不公平比较** | 自己模型高分辨率、baseline 低分辨率 | 同设置对照 (9.4-L3 控混淆) |
| **prompt 不对等** | 给自己模型精调 prompt, baseline 用默认 | 同 prompt 协议 (9.4-L3) |
| **没有方差** | 单次评测当结论 (VLM 也有随机性) | 多次/多 prompt 报方差 (9.4-L5) |

> 这张表是 9.3 攻击清单 + 9.4 公平对照 + 9.6 诚实图在 VLM 评测的具体化。**审一篇 VLM 论文时, 逐条查这些; 写自己的 VLM 结果时, 逐条避。** 你 Module 9 的诚实工具链在多模态全部复用。

---

## 2. 一个 VLM 特有的隐蔽陷阱: 评测 prompt 的敏感性

VLM 对**怎么问**极其敏感:
- 同一张图、同一个意思, 换个问法 (「有几个?」vs「数一下」vs「count the objects」) 准确率可能差很多。
- 选项顺序、是非题的「Yes 在前还是 No 在前」都影响结果 (位置偏置)。

> 这意味着一个 VLM 的「benchmark 分数」可能很大程度是**prompt 工程的产物**, 而非真实能力。诚实的报告要: ① 固定并公开评测 prompt; ② 报告对 prompt 变体的鲁棒性 (换几个问法看分数稳不稳)。这是 VLM 评测的「方差」维度 (9.4-L5 的多模态版)。

---

## 3. 诚实报告 VLM 结果 (checklist)

把你 9.5/9.6 的诚实纪律用在 VLM:
- [ ] **多维剖面**: 报感知/推理/OCR/定位/幻觉各维度, 不只一个总分 (9.6 一图一信息)。
- [ ] **抗幻觉指标**: 必报 POPE/CHAIR 类幻觉指标。
- [ ] **公平对照**: 同分辨率、同 prompt 协议、同评测集 (9.4-L3)。
- [ ] **方差**: 多 prompt/多次评测报 error bar (9.4-L5)。
- [ ] **污染声明**: 说清 benchmark 是否可能被训练见过 (9.3 Q4)。
- [ ] **定性样例诚实**: 给随机 (非 cherry-picked) 样例, 含失败案例 (9.6-L4 / 9.7 limitations)。
- [ ] **prompt 公开**: 公开评测 prompt, 报 prompt 鲁棒性。

> 你的 N2 会用 9.6 的 `plotstyle` 把 POPE 评测结果画成出版级图 —— 多维、带误差棒、诚实。**VLM 评测的诚实报告 = M6 评测 + 9.4 实验严谨 + 9.6 诚实图 + 本专题幻觉, 全部叠加。**

## 4. M10.6 收口: 会造也要会测

```
   M10.1-10.5: 造 VLM (读/画/听/说)
   M10.6: 测 VLM —— 它说得对吗? 真看图吗? 会幻觉吗?
   ───────────────────────────────────────
   会造 + 会测 = 完整的 VLM 能力 (造出来还能诚实地知道它行不行)
```

> 一个只会造不会测的人, 造出的 VLM 自己都不知道行不行 (甚至被幻觉骗了还以为很强)。**评测是 VLM 工程的「良心」** —— 它让你诚实地知道模型的真实能力和缺陷。这呼应 9.5/9.6 的诚实主线: 不自欺, 才能真正进步。

## 5. 本讲小结 (M10.6 收口) + 通往 M10.7

- VLM 评测特有陷阱: 污染 / 挑样例 / 不报幻觉 / 单一总分 / 不公平 / prompt 不对等 / 没方差 —— 用 9.3+9.4+9.6 逐条避。
- **prompt 敏感性**是 VLM 隐蔽陷阱: 分数可能是 prompt 工程产物 → 固定+公开 prompt, 报鲁棒性。
- **诚实报告 checklist**: 多维剖面 + 抗幻觉 + 公平对照 + 方差 + 污染声明 + 诚实样例 + prompt 公开。
- 会造也要会测; 评测是 VLM 工程的良心 (接 9.5/9.6 诚实)。
