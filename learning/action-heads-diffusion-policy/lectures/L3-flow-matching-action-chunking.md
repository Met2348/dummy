# L3 · flow-matching 动作头 + action chunking

> 22-min lecture · 目标: 理解 π 的两个关键工程 —— flow-matching 少步出动作 (接 M13.2) + action chunking (一次预测一段动作)。

---

## 0. 从「能多峰」到「能实用」

L1/L2 让扩散动作头能表达多峰。但要真在机器人上用, 还差两个工程: ① 够快 (高频控制) ② 够稳 (不抖、不短视)。π 系列的答案是 **flow-matching** (解决快) + **action chunking** (解决稳)。

---

## 1. flow-matching 动作头 (接 M13.2): 解决「快」

L2 说朴素扩散采样要多步 (50 步), 机器人高频控制吃不消。**flow-matching** (你的 M13.2) 用更直的概率路径, **几步就能出动作**:

- π 用 flow-matching 当动作头, 少步采样 → 50Hz 高频控制可行。
- 机制就是你 M13.2: 学速度场, ODE 少步积分 (rectified flow 把路径拉直)。

> 这是 M13.2 在机器人上的直接落地: **少步采样不是优化, 是高频控制的硬需求**。π 选 flow-matching 而非朴素扩散, 就是为了这个「快」。你 M13.2 学的拉直/少步在这里是命脉。

---

## 2. action chunking: 解决「稳」

**action chunking** = 一次预测**一段**动作 (H 步), 而非单步:

```
   单步 (chunk=1):  每步重新预测一个动作  → 可能步步抖 (多峰下来回切模式)
   chunking (H>1):  一次预测 H 步动作, 开环执行, 再重规划  → 一段内连贯
```

为什么需要 chunking:
- **抗抖动 (模式切换)**: 多峰策略每步独立采样, 可能这步采"上绕"、下步采"下绕" → 来回横跳 (dithering)。chunking 一次定一段, 锁住一个模式。
- **平滑/连贯**: 一段动作一起生成, 时间上更连贯。
- **抗延迟**: 预测一段, 给推理/通信留缓冲 (真实机器人有延迟)。

> 你 N2 会做 chunk 大小的消融。注意 chunking 是**双刃剑** (见 L3 §3)。

---

## 3. action chunking 的代价: 反应性 (你 N2 会量到)

chunk 越大越连贯, 但**越不能闭环纠错**:

```
   chunk 小 (1):  反应快 (每步看新状态纠错), 但易抖
   chunk 大 (8):  连贯/抗抖, 但开环执行一长段 → 看不到中途状态变化 → 反应慢、易偏
```

> 这是 **平滑/连贯 vs 反应性 (闭环纠错)** 的核心权衡。你 N2 会在玩具上量到: chunk 增大, 块内更连贯, 但**成功率可能下降** (开环执行一长段, 失去闭环纠错, 在需要随时纠偏的任务上吃亏)。真实系统折中 (常配合 temporal ensembling 等技巧)。**没有免费的 chunk; 大小要按任务的「需要多频繁纠错」来定。**

---

## 4. π 的完整动作头 = flow + chunking + 高频

把 π 的动作头拼起来:
```
   观测 → VLM backbone (M10) → 特征
        → flow-matching 动作头 (M13.2): 少步生成
        → 一次出一个 action chunk (H 步)
        → 高频执行 (50Hz) + 周期重规划 (闭环纠错)
   = 多峰 + 连续 + 平滑 + 快 + 稳
```

> 这就是 SOTA 灵巧操作 VLA 的动作头全貌。每一块你都学过: VLM (M10)、flow-matching (M13.2)、扩散/条件 (M13.1/13.3)、chunking 权衡 (本讲)。**π 的动作头 = 你已学部件的组合 + 工程权衡。**

---

## 5. 本讲小结 + 通往 L4

- **flow-matching 动作头** (接 M13.2): 少步出动作 → 高频控制可行 (解决「快」)。
- **action chunking**: 一次预测 H 步 → 抗抖动/平滑/抗延迟 (解决「稳」)。
- chunking 是双刃剑: **平滑/连贯 vs 反应性 (闭环纠错)** 权衡 (N2 量到, 大 chunk 可能掉成功率)。
- π 动作头 = VLM + flow-matching + chunking + 高频, 全是你学过的部件。
