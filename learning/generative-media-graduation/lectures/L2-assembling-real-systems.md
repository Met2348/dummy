# L2 · 装配: 把部件拼成真实系统

> 22-min lecture · 目标: 把「前沿系统 = 部件组合」做实 —— 拆解 SD / Sora / 世界模型 / dLLM 各是哪些 M10/M13 部件的组合。

---

## 0. 装配 = capstone 的核心技能

研究/工程的真本事不是记住单个技术, 而是**看到一个新系统能把它拆回已知部件**。本讲带你拆四个真实系统, 每个都标注「这块来自你哪个专题」。拆完你会确信: **你已经懂了它们的每一块。**

---

## 1. 装配 Stable Diffusion (文生图)

```
   "a red cat"
     → 文本编码器 (CLIP 文本塔)            ← M10.1
     → 噪声 latent (VAE 编码空间)          ← M13.3-L2 latent
     → DiT/U-Net 去噪 T 步:                ← M13.3-L1 DiT
          cross-attend 文本               ← M10.2 cross-attn
          CFG 放大文本方向                 ← M13.3-L3 CFG
     → VAE 解码 → 图                       ← M13.3-L2
```
> SD = 5 个你学过的部件拼起来。没有一块是黑魔法。

---

## 2. 装配 Sora (文生视频)

```
   "a cat surfing"
     → 文本编码                            ← M10.1
     → 噪声 latent 视频 (3D VAE)           ← M13.3-L2 的视频版
     → 切时空 patch                        ← M10.5-L1 时空 token
     → DiT 时空去噪 (cross-attend 文本+CFG) ← M13.3 + M13.4-L1 时空扩散
     → 3D VAE 解码 → 视频                   ← M13.4-L2
```
> Sora = SD + 时间维 (时空 patch + 时空扩散)。多的那一维你在 M13.4 学过。

---

## 3. 装配交互式世界模型 (可玩世界)

```
   当前帧 + 动作 a
     → (隐式) 视频生成底座                  ← M13.4 视频扩散
     → 动作当条件注入 (token/cross-attn)    ← M13.3-L4 条件 + M10.2
     → 预测下一帧 (动作条件续帧)            ← M13.5-L3 可玩世界
     → 持久状态/记忆 (对抗长程漂移)         ← M13.5 (open, gap)
```
> 世界模型 = 视频模型 + 动作条件 + (理想的) 记忆。前两块你会, 第三块是 gap (L3 找)。

---

## 4. 装配 dLLM (文本扩散)

```
   部分文本 (含 [MASK])
     → 双向 transformer (无因果掩码)        ← 你的本行 transformer
     → 预测被遮位 (masked CE 训练)          ← M13.6-L1 masked diffusion
     → 置信度并行迭代解码                    ← M13.6-L1 (MaskGIT 思想)
     → 可固定约束位做可控生成               ← M13.3 条件思想 + M13.6-L3
```
> dLLM = transformer (本行) + 离散扩散范式。地基你全有。

---

## 5. 装配的元技能: 怎么拆任何新系统

四个系统拆完, 提炼一个**通用拆解流程** (带走的元技能):

1. **它生成什么 / 预测什么?** (信念一: 生成=预测)
2. **数据/状态是什么维度?** (图/视频/文本/状态-动作)
3. **去噪/预测网络是什么?** (U-Net / DiT / 双向 transformer)
4. **在什么空间?** (像素 / latent / 离散 token)
5. **条件怎么注入?** (cross-attn / token / CFG)
6. **怎么省成本?** (latent / patch / 少步 / 分解注意力)
7. **哪里还没解决?** (gap → 研究机会, L3)

> 拿这 7 问去拆任何 2026 的新生成系统 (你没见过的), 你都能在十分钟内把它还原成已知部件 + 找到它的创新点和 gap。**这就是 capstone 给你的真正能力: 不是记住系统, 是会拆系统。**

---

## 6. 本讲小结 + 通往 L3

- 四个真实系统全部拆成 M10/M13 部件: SD (5块) / Sora (SD+时间维) / 世界模型 (视频+动作+记忆) / dLLM (transformer+离散扩散)。
- 没有一块是黑魔法; 你已懂每一块, 前沿只是组合 + 规模 + (少数) open gap。
- 元技能: 7 问拆解流程, 能还原任何新生成系统。

> **下一讲 L3「找 gap」**: 拆解时冒出的「哪里还没解决」就是研究机会。用你 M9 的找 gap 框架, 在 M13 全链上系统地挖研究问题, 产出 idea 卡。

**动手**: 在 N1 里你已驱动了全栈。现在挑一个你**没在本课见过**的生成系统 (如某个新文生视频模型), 用本讲的 7 问拆它一遍, 看能还原多少。
