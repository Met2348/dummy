# L1 · Module 13 全链回顾: 一张图串起生成式媒体

> 22-min lecture · 目标: 把 M13 六个专题串成一条线, 提炼贯穿全链的核心信念。Capstone 开篇。

---

## 0. 你这一路造了什么

M13 从「不懂扩散」走到「懂 Stable Diffusion / Sora / 世界模型 / dLLM 怎么造」。回头看全链:

```
  13.1 扩散地基      前向加噪 ⇄ 反向去噪 (预测噪声=预测 score)
   │
  13.2 流匹配/加速    更直的概率路径, 少步采样 (一致性模型)
   │
  13.3 DiT+latent+CFG transformer 当去噪器 + 压缩空间 + 条件控制  ← Stable Diffusion 全部件
   │
  13.4 视频          加时间维, 时空扩散, 时序连贯  ← Sora = 时空patch+DiT+latent
   │
  13.5 世界模型       加动作条件, 学环境动态, 脑内想象  ← 生成↔决策的枢纽
   │
  13.6 dLLM          文本也能扩散, 双向并行解码  ← 绕回 NLP 本行
```

> 这不是六个孤立专题, 是**一条加维度的线**: 图 → (加速) → (骨架/条件) → +时间 (视频) → +动作 (世界模型) → 换模态 (文本)。每一步都复用前面的机制。

---

## 1. 贯穿全链的核心信念 (三条)

**信念一: 生成 = 学会预测。**
- 扩散预测噪声、视频预测下一帧、世界模型预测下一状态、dLLM 预测被遮 token。
- 全都是「给定部分, 预测其余」。**强预测目标 + 海量数据 → 涌现的生成/世界知识** (M13.5-L2 的 Sora 洞察 = LLM 预测下一词的同一故事)。

**信念二: 前沿系统 = 已学部件的组合 + 规模。**
- Stable Diffusion = 文本塔(M10.1) + cross-attn(M10.2) + DiT + CFG + latent。
- Sora = 时空 patch(M10.5) + DiT + latent + 时空扩散。
- 你的 toy 和它们**只差规模** (同 M10.7 mini-VLM)。

**信念三: 架构 = 成本/约束的优化结果。**
- latent (省算力)、时空 patch (统一表示)、分解注意力 (省二次方)、少步采样 (省推理) —— 每个设计都在解一个成本/约束 (M13.4-L4)。

> 这三条信念不只对 M13 成立, 是整个课程的底层操作系统。带着它们, 你能拆解任何没见过的前沿系统: 问「它预测什么 / 由哪些部件组成 / 在优化什么成本」。

## 2. 全链的「机制守恒」: 变的与不变的

| 专题 | 变 (加了什么) | 不变 (复用什么) |
|---|---|---|
| 13.1→13.2 | 概率路径更直 | 去噪/预测目标 |
| 13.2→13.3 | 去噪网络换 transformer + 条件 | 扩散框架 |
| 13.3→13.4 | 数据加时间维 | DiT + latent + 扩散机制 |
| 13.4→13.5 | 加动作条件 | 生成式预测 |
| 13.5→13.6 | 连续→离散 (文本) | 扩散 (遮盖=加噪) + transformer |

> 看这张表你会发现: **每一步只变一点, 其余全复用**。这就是为什么你能一路走下来 —— 不是学六个新东西, 是把一个东西加六次维度。capstone 的价值就是让你看清这个「机制守恒」。
