# L3 · 找 gap: 在生成式媒体里挖研究问题

> 22-min lecture · 目标: 用 M9 的找 gap 框架, 在 M13 全链系统地挖研究问题, 产出可执行的 idea 卡。

---

## 0. 从「懂系统」到「推进系统」

L2 教你拆系统 (还原成部件)。研究者还要再进一步: **找到系统没解决的地方, 提出推进它的想法**。这正是你 M9「科研技能」学的找 gap, 现在用在 M13 这片热土上。

---

## 1. gap 从哪来 (四个来源)

回顾 M9.3 的找 gap 框架, 在 M13 全链上的具体来源:
- **拆解时的「这块还没解决」** (L2 第 7 问): 如世界模型的持久记忆。
- **每个专题 lecture 标的 gap**: 视频长程一致 (M13.4-L3)、物理正确性 (M13.5-L4)、dLLM 加速 (M13.6-L4)。
- **跨专题的「能不能迁」**: M13.2 少步采样能否迁到视频/dLLM? AR 的 RLHF 能否迁到 dLLM?
- **评估的滞后**: 生成越强, 评估越是瓶颈 (视频物理评测、世界模型「懂vs拟合」、dLLM 似然评估)。

> 注意一个高产模式: **「把 A 世界的成熟技术迁到 B 世界」**。少步采样 (扩散)→dLLM、KV cache (AR)→dLLM、RLHF (AR)→dLLM、域随机化 (sim2real)→世界模型。范式迁移是 idea 富矿, 因为两边你都懂。

---

## 2. M13 的 gap 雷达 (7 个, N2 会逐个产出 idea 卡)

`generative_capstone.py` 的 `GAPS` 整理了 7 个 M13 研究 gap:

1. **少步高质量 dLLM 解码** ← M13.2 一致性 + M13.6
2. **视频长程一致 / 持久记忆** ← M13.4 + M13.5
3. **世界模型物理正确性评测** ← M13.5 + 评估
4. **dLLM 对齐迁移 (RLHF→dLLM)** ← M13.6 + 你的 RL/DPO
5. **统一生成 (跨模态一个扩散模型)** ← M13.3 + M13.6 + M10.5
6. **实时视频 (少步+蒸馏)** ← M13.2 + M13.4
7. **世界模型 × 具身 sim2real** ← M13.5 + M11

> 这些不是凭空列的, 每个都标了「连接哪些专题」—— 因为**好 gap 长在两个你都懂的领域的交界**。对你 (NLP 背景), gap 1/4/5 (都涉及 dLLM) 是低成本高赔率的切入点。

---

## 3. 一张好 idea 卡的四要素 (接 M9)

`make_idea_card` 把每个 gap 结构化成 idea 卡, 四要素:
1. **问题 (gap)**: 具体到「什么没解决」, 不空泛。
2. **为什么难**: 不是没人做, 是有真障碍 (否则早被做了)。
3. **最小可验证实验**: 在 toy 上能跑的第一步 (不是「训个大模型」)。
4. **连接**: 复用你哪些已有知识 (降低执行成本)。

> 关键是**最小可验证实验**: 好 idea 卡不止说「该研究 X」, 而是给出「我明天能在玩具上跑的第一个实验」。这是 M9 反复强调的: 从 idea 到可执行的第一步。你 N2 产出的每张卡都带这一栏。

---

## 4. 怎么挑你该做的 gap (匹配自己)

7 个 gap 不是都适合你。挑选准则 (接 M9 的研究品味):
- **低地基成本**: 选你已有大半地基的 (你 NLP 背景 → dLLM 相关 gap 1/4/5)。
- **有 toy 入口**: 选能在小规模验证的 (本课所有 gap 都配了 toy 实验)。
- **交界优势**: 选你比别人多懂一边的 (你懂 NLP+扩散 → dLLM 范式迁移你有优势)。
- **赔率**: 新但已证明可行的方向 (dLLM: 够新有红利, 已证明可 scale 不是空中楼阁)。

> 对你这个 2026 秋入学、NLP/LLM 方向的博 0: **dLLM 的范式迁移 (gap 1/4/5) 是量身定做的切入点** —— 你的 transformer + 扩散 (M13) + RL/对齐知识全用得上, 方向新有红利, 且有 toy 能立刻上手。

---

## 5. 本讲小结 + 通往 L4

- gap 四来源: 拆解残留 / 专题标注 / 跨专题迁移 / 评估滞后; **范式迁移是富矿**。
- M13 gap 雷达 7 个, 每个标「连接哪些专题」(好 gap 长在交界)。
- idea 卡四要素: 问题 / 为什么难 / **最小可验证实验** / 连接。
- 挑 gap 准则: 低地基成本 + toy 入口 + 交界优势 + 赔率; 对你 dLLM 相关最匹配。

> **下一讲 L4「毕业 + 路线」**: M13 在 2026 的位置, 它和 M11 (具身, 共享世界模型) / M12 (可解释, 理解这些生成模型) 怎么连, 以及给你这个研究者的下一步路线。
