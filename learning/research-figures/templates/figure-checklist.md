# 图 checklist: 图 ___ (____________)

## 一图一信息 (L1)
- [ ] 能用一句话说出这张图要让读者看到的**那一个**结论: ____
- [ ] 没塞多余信息 (≤1 个主信息)

## 清晰 (L1/L2)
- [ ] 图类型选对 (趋势→线 / 比较→柱 / 分布→误差棒 / 矩阵→热力图)
- [ ] data-ink: 去掉上右边框、3D、重网格、装饰
- [ ] 字号: 缩到目标栏宽 (~3.3in) 后仍读得清 (7-9pt)
- [ ] 颜色: 色盲安全 (Okabe-Ito/tab10); 顺序数据用 viridis 非 jet; 灰度也能区分
- [ ] 同一对象在所有图里同色

## 诚实 (L4) ⚠ 用 9.3 攻击清单自查
- [ ] 柱状图 y 轴从 0 开始 (不截断夸大)
- [ ] 含均值的图带误差棒, 且图注注明是什么 (std/SEM/CI + 几种子)
- [ ] 没挑有利区间 / 没双轴障眼 / 没 cherry-pick 样例
- [ ] 视觉冲击匹配效应真实大小 (不显著的差距不画得很大)

## 格式 (L2)
- [ ] 导出矢量 PDF (线条图) + 高 DPI PNG (预览)
- [ ] figsize 匹配栏宽; 多子图用 (a)(b)(c) + constrained_layout

## 图注 + 呼应 (L4)
- [ ] 图注 self-contained: 首句是结论, 写清设定/误差棒, 给读图指引
- [ ] 这张图对应正文一个 claim, 正文引用并解读了它
