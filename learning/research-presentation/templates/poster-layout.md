# Poster 布局: ____________

## 设计原则 (自检)
- [ ] 一个**巨大的核心图** 当视觉中心 (路过被它抓住)
- [ ] 大字号 (标题远处可读, 正文 1 米可读)
- [ ] 极少文字 (比 slide 还少; 要点 + 图)
- [ ] 明确动线 (编号/箭头/分栏引导阅读顺序)
- [ ] 留白 (别填满)
- [ ] takeaway 大且醒目 (路人带走的唯一信息)

## 布局 (Z 字 / 栏式动线)
```
┌────────────────────────────────────────────┐
│  大标题 + 作者 + 机构 (远处可见)               │
├──────────────┬──────────────┬───────────────┤
│ ① 问题/动机   │ ③ 方法 (大图!) │ ⑤ 关键结果     │
│ ② gap        │ ④ 核心 idea    │   (那张交互图)  │
├──────────────┴──────────────┴───────────────┤
│  ⑥ TAKEAWAY (一句话, 最大最醒目)               │
└────────────────────────────────────────────┘
```

- ⑥ takeaway: ____

## poster session 玩法
- 准备好 1-2 分钟 pitch (见 pitch-ladder.md) 当口头讲解
- 目标不是让人读完, 是**引发对话** (拿反馈/找合作/被记住)
