# L4 · 干预的严谨性: 干预是消融实验的极致

> 20-min lecture · 目标: 理解怎么设计严谨的干预实验 (接 M9.4), 不被假象骗。收口 12.3。

---

## 0. 干预 = 消融实验的极致

你 M9.4 学过消融实验 (ablation study): 去掉一个组件看效果。mech interp 的 activation patching / ablation 是它的**极致版** —— 不是去掉一个模块, 而是精确干预到**单个激活**。所以 M9.4 的所有严谨性原则在这里**加倍重要**。本讲讲怎么做严谨的干预。

---

## 1. 严谨干预的四要素 (接 M9.4)

- **单变量**: 一次只 patch/ablate 一个东西 (否则分不清谁的效应)。
- **对照**: clean/corrupt 要尽量像, 只差你关心的信息 (L2 的灵魂)。
- **度量明确**: 用定义清晰的指标 (logit difference / 恢复率), 不是「看着像」。
- **分布内**: 干预后的激活要尽量留在正常分布 (用 mean/resample 而非 zero, L3)。

> 这四条直接来自 M9.4 的实验设计。**activation patching 就是一个内部的消融实验**, 严谨性标准完全一样。不严谨的 patching (多变量、对照不干净、zero ablation 出分布) 会给你漂亮但错误的因果故事。

---

## 2. 陷阱一: 分布外干预 (最隐蔽)

L3 提过: zero ablation 把激活推到 0, 可能远离正常分布 → 模型崩溃, 但不是因为组件必要, 而是因为输入异常。

> 诊断: 如果一个 ablation 让模型「完全乱套」(而非「特定能力下降」), 警惕是 OOD 假象。解药: 用 **resample ablation** (替换成另一个真实样本的激活) 或 **mean ablation** —— 这些值是模型见过的, 不出分布, 干预效应更可信。**「模型崩了」≠「这个组件重要」。**

---

## 3. 陷阱二: 把恢复率当全部

patching 的恢复率高很诱人, 但要小心:
- **冗余**: patch 位置 A 恢复了, 不代表 A 是唯一路径 (B 可能也能)。要配 ablation 查必要性。
- **非线性交互**: 单独 patch 各组件都没用, 但一起 patch 就恢复 (组件协作)。单组件 patching 会漏掉这种 circuit。
- **度量依赖**: 换个度量 (logit diff vs 概率 vs 准确率) 结论可能变。报告时说清用了什么度量。

> 严谨的 interp 不止报一个恢复率, 而是: **充分 (patching) + 必要 (ablation) + 度量稳健 (多指标) + 分布内 (温和干预)** 多重验证。这才是「定罪」而非「讲故事」。

---

## 4. 陷阱三: 过度解读单个实验

- 一对 clean/corrupt 的结果可能是巧合 (特定输入)。要在**多个样本**上重复, 看效应稳不稳 (统计, 接 M9.4)。
- 玩具上干净的因果, 大模型上可能更乱 (冗余多)。别过度外推 (M12.1-L4)。

> 好的 interp 实验: **多样本 + 报告方差 + 多度量 + 充要双验证 + 诚实说边界**。你的 EE/实验背景在这里直接变研究质量。**mech interp 的可信度全在干预的严谨性上** —— 这是它区别于「电路占星术」的根本。

---

## 5. 严谨干预 checklist (带走)

做任何 patching/ablation 实验, 自检:
1. **单变量**? (一次一个干预)
2. **对照干净**? (clean/corrupt 只差关心的信息)
3. **度量明确**? (logit diff / 恢复率, 不是看着像)
4. **分布内**? (mean/resample 而非 zero)
5. **充要双验证**? (patching 充分 + ablation 必要)
6. **多样本 + 报告方差**?
7. **诚实边界**? (玩具≠真实, 冗余路径)

> 拿这 7 条做 interp, 你的因果结论才站得住。这是 mech interp 的研究质量底线 (接 M9.4 + M9.3)。

---

## 6. 本讲小结 (12.3 收口) + 通往 12.4

- 干预 = 消融实验的极致 (接 M9.4); 四要素: 单变量/对照/度量/分布内。
- 陷阱: 分布外干预 (zero ablation 假象, 用 mean/resample) / 把恢复率当全部 (查冗余+必要) / 过度解读单实验 (多样本)。
- 严谨干预 7 问 checklist; 因果可信度全在干预严谨性 (区别于电路占星术)。
