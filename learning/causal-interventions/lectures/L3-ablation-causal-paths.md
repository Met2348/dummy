# L3 · ablation 与因果路径: 找最小必要子集

> 22-min lecture · 目标: 掌握 ablation (置零干预), 用它找「哪些组件是必要的」, 连成因果路径。

---

## 0. patching 找「充分」, ablation 找「必要」

L2 的 denoising patching 找「patch 这里**足以**恢复行为」(充分性)。**ablation** 问反面: 「去掉这里, 行为**会坏吗**?」(必要性)。两者合起来才能定位**最小充要的因果子集**。

---

## 1. ablation 怎么做

```
   ablation = 把某 (层, 位置/组件) 的激活"删掉", 看行为损害多少
   "删掉"的方式:
     - 置零 (zero ablation): 激活设为 0
     - 均值替换 (mean ablation): 设为数据集均值 (更温和, 推荐)
     - 随机替换 (resample ablation): 设为另一个样本的激活
   损害大 → 这个组件对行为必要; 损害小 → 不必要 (可去掉)
```

> 你 N2 会对玩具的每个 (层, 位置) 做 ablation, 测对答案的损害, 扫出**最小必要子集** (哪些组件去掉就坏)。注意 ablation 方式有讲究: **zero ablation 可能太暴力** (把激活推到分布外, 制造假象), mean/resample ablation 更接近「正常但无此信息」, 结论更可信 (L4 会讲严谨性)。

---

## 2. 因果路径: 把组件连成链

单个组件的因果效应不够, 要看**信息怎么流**: 组件 A 的输出 → 被组件 B 读 → B 影响输出。这就是**因果路径 (causal path)**:

```
   path patching: 只 patch「A → B 这条边」上的激活, 看 B 是否受影响
   → 区分「A 直接影响输出」vs「A 通过 B 影响输出」
   → 把单个组件的效应, 细化成"信息沿哪条路走"
```

> 这把孤立的「哪个组件重要」升级成「信息流图」: A 写入某 feature → B 读它 → 算出结果。多条这样的边连起来 = **circuit** (M12.5)。ablation 找节点 (必要组件), path patching 找边 (信息流), 合起来画出电路。

---

## 3. 相关 / 充分 / 必要: 三层证据

把 M12.2-12.3 的证据强度排个序:
| 证据 | 工具 | 强度 |
|---|---|---|
| 相关 | 探针/logit lens/attention | 弱 (可能旁路) |
| 充分 | denoising patching (patch 它就恢复) | 中 |
| 必要 | ablation/noising (去它就坏) | 中 |
| 充要 | patching + ablation 都指向同一组件 | 强 |

> 最强的因果主张: **充分 + 必要都成立** (patch 它能恢复, 去它就坏)。只有充分可能有冗余 (别处也能), 只有必要可能不够 (它必要但不单独决定)。**严谨的 interp 要两头验证。** 你 N1 (patching=充分) + N2 (ablation=必要) 合起来就是这个充要论证。

---

## 4. 干预的「分布外」陷阱 (重要警示)

干预 (尤其 zero ablation) 有个隐患: **把激活改成模型从没见过的值, 制造分布外 (OOD) 行为**:
- zero ablation 把激活推到 0 (可能远离正常激活分布) → 模型乱套, 但不是因为「这个组件必要」, 而是因为「输入异常」。
- 假象: 你以为定位了重要组件, 其实只是把模型搞崩了。

> 解药: 用**温和的干预** (mean/resample ablation: 替换成「正常但无关」的值, 不出分布)。这是 L4 (干预严谨性) 的核心。**干预要尽量"只改你关心的信息, 不制造异常"** —— 否则因果结论不可信。

---

## 5. 本讲小结 + 通往 L4

- **ablation = 删掉某激活看损害** (置零/均值/重采样); 损害大=必要组件。
- **因果路径 / path patching**: 区分直接 vs 间接影响, 把组件连成信息流 (→ circuit M12.5)。
- 证据三层: 相关 (弱) → 充分 (patching) / 必要 (ablation) → **充要** (两者都指向, 强)。
- 陷阱: zero ablation 可能制造**分布外**假象 → 用 mean/resample ablation (L4)。

> **下一讲 L4「干预的严谨性」**: 干预就是消融实验的极致 (接 M9.4) —— 怎么设计干预才严谨 (对照/度量/分布内/统计), 不被假象骗。

**动手**: 去 `N2`, 对玩具做 ablation 扫描, 找最小必要子集; 对比 N1 的 patching (充分) —— 两者都指向同一位置 = 充要因果证据。
