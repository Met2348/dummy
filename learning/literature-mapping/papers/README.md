# papers/ — literature-mapping 参考源

本专题是**方法/工作流**类。papers/ 收「文献综述方法」+「本专题示范用的偏好优化子领域」两类。

## 文献综述 / 滚雪球方法论
- **Guidelines for snowballing in systematic literature studies** — C. Wohlin (2014). 滚雪球检索的方法论原始文献, L2 的来源。
- **Systematic vs scoping reviews** — 各类方法学指南 (如 PRISMA / PRISMA-ScR)。L1 区分三类综述的依据。
- Keshav, *How to Read a Paper* (见 9.3 papers/, 三遍读法在摸领域时反复用)。

## 示范子领域: 偏好优化 (preference optimization)
本专题工具内置的离线数据集围绕这个子领域 (对应你的 `learning/dpo-family` 复现)。代表论文:
- InstructGPT (RLHF) — Ouyang et al. 2022
- DPO — Rafailov et al. 2023 (本子领域的奠基/中心节点)
- IPO / KTO / ORPO / SimPO — 2023-2024 直接偏好优化家族
- 一篇该子领域的 survey (搜 "preference optimization survey" / "alignment survey")
