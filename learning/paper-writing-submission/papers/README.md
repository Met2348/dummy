# papers/ — paper-writing-submission 参考源

本专题教**写作装配 + 投稿流程**。写作方法论本身见 `how_to_write_a_paper` 技能包, 这里收投稿/评审/rebuttal 的参考。

## 写作 (复用, 不重复)
- **`how_to_write_a_paper/` 技能包** (本仓库) — 叙事先行方法论 + 逐部分写作细则 + 图表公式 + 审稿人视角清单。**写每一节时打开它。**
- (经典外部参考) "Writing for Computer Science" — Justin Zobel; "The Elements of Style" — Strunk & White。

## 投稿 / 评审 / rebuttal (本专题新增)
- 各顶会官网 Call for Papers (ACL/EMNLP/NeurIPS/ICML/ICLR/COLM) — 最权威的 deadline/格式/政策来源, 每年更新。
- **ACL Rolling Review (ARR)** 文档 — 滚动审稿机制 (L2)。
- OpenReview 使用文档 — 提交与 discussion/rebuttal 机制 (L2/L3)。
- "How to Write a Good Rebuttal" 类社区指南 (各课题组/资深研究者博客, 用 9.3 批判式读, 取其共识)。
- NeurIPS / ACL Reviewer Guidelines — 反过来读"审稿人被要求看什么", 帮你预判攻击点 (也接 9.9 审稿)。

## 学术诚信 (L4)
- 各会的 ethics policy / dual-submission policy / plagiarism policy。
- ACM/IEEE 学术行为准则。

## 为什么 papers/ 这么轻
本专题知识在**可跑工具** (`paper_assembler` / `rebuttal_kit`)、课件流程、以及你已有的写作技能包里。
真练习: 拿你一个真实复现去走一遍「装配→自检→(模拟)投稿→(模拟)rebuttal」。
